CONS∃NSUS

This is the software for Consensus in Organizations study with Anna Popova (University of Illinois) and Michel Regenwetter (University of Illinois). It is programmed in Matlab, and compiled to use as a (somewhat) standalone application. This work was supported by National Science Foundation grants CCF # 12-16016 and SES # 08-20009 (PI: M. Regenwetter) and the University Library at the University of Illinois at Urbana-Champaign (PI: A. Popova).

  • SocialChoice.exe, the CONS∃NSUS program itself, current as of September 13, 2012 (license file inside).
  • A tutorial that explains the controls and conventions of the program in Word and PDF.
  • MCRInstaller.exe v7.15, the Matlab libraries necessary for running SocialChoice.exe in case you don't have Matlab installed. My right to distribute these libraries is limited by a license agreement with MathWorks, Inc, and you can only use them to run Social Choice software. You have to agree to MCR Library License to use these libraries to run Social Choice.
  • The Matlab code that can be compiled into the CONS∃NSUS program or run separately, current as of September 13, 2012 (license file inside).
